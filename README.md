🚀<b>Projeto Starter</b>

É um projeto de uma pagina desenvolvida em HTML com CSS e futuramente com funções JavaScript.
 
📋🔧Como acessar a aplicação?:<br>
Sendo uma pagina HTML voce pode acessar diretamente pela URL: https://richardfelicetti.github.io/projetostarter/
<br>Mas caso queria subir ele em sua maquina:<br>
*Faça o donwload dos repositorios;<br>
*Abra ele pelo "index.html";<br>
*Caso nao carregue o CSS do site, recomendo utilizar alguma ferramenta de serverlocal (por exemplop plugin "Live Server" do VScode).
<br><br>
⚙️ Executando os testes:<br>
Como a aplicação esta somente em HTML/CSS, os testes são somente de interface nada referente a funções. 

📦 Desenvolvimento:<br>
Inicialmente esta desenvolvido somente a interface, mas será futuramente atualizado com funcoes de JavaScript resolvendo uma API para alterar os preços dos planos com respectivos ciclos de pagamento.
